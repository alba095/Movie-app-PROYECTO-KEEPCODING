# Movie App

Aplicación web desarrollada con Flask que permite buscar películas mediante la API de OMDb, visualizar su detalle, añadir comentarios y puntuar cada película del 1 al 5 mostrando el promedio en estrellas.

## Tecnologías
- Python
- Flask
- SQLite
- Bootstrap 5
- OMDb API

## Funcionalidades
- Búsqueda por título y año
- Vista de detalle con información completa
- Sistema de comentarios con fecha automática
- Sistema de calificaciones con promedio en estrellas
- Control de errores y validaciones

## Ejecución
1. Crear entorno virtual
2. Instalar dependencias: `pip install flask requests`
3. Configurar API key de OMDb
4. Ejecutar: `python app.py`
5. Abrir `http://localhost:5000`
